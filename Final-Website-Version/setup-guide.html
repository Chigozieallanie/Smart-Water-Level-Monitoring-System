Smart Water Monitoring System

Complete setup guide for ThingSpeak integration with ESP32 and automatic SMS reporting

1
Create ThingSpeak Account & Channel

Visit ThingSpeak.com and create a free account.

  • Create a new channel with 5+ fields
  • Field 1: Distance (cm)
  • Field 2: Water Level (liters)
  • Field 3: Depth Percentage (%)
  • Field 4: Immediate Commands
  • Field 5: Analysis Commands

Save your Channel ID, Write API Key, and Read API Key.

2
Configure Website Dashboard

Open your dashboard and navigate to the ThingSpeak Configuration section.

  • Enter your ThingSpeak Channel ID
  • Enter your Read API Key
  • Enter your Write API Key
  • Click "Save ThingSpeak Configuration"
3
Setup Automatic Reporting

Configure when you want to receive SMS reports:

  • Daily Reports: Choose time for daily water level summaries
  • Weekly Reports: Select day and time for weekly analysis
  • Custom Reports: Set custom intervals (hours)

Click "Save Schedule Configuration" to activate.

4
Update Arduino Code

Update your ESP32 code with your ThingSpeak API key:

String str="GET https://api.thingspeak.com/update?api_key=YOUR_WRITE_API_KEY&field1=" + String(distance) +"&field2="+String(ltr)+"&field3="+String(depth);

Replace YOUR_WRITE_API_KEY with your actual Write API Key.

5
Test the System

Use the manual controls to test your setup:

  • Click "Send Test SMS" to verify SMS functionality
  • Click "Send Manual Alert" to test emergency notifications
  • Click "Refresh Data" to update dashboard with latest readings

Important Notes

  • Keep your browser tab open for automatic scheduling to work
  • The system stores configuration in your browser's local storage
  • For 24/7 operation, consider hosting on a server or using GitHub Pages
  • SMS alerts are sent via your ESP32 GSM module, not the website
